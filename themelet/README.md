# Model Context Polytechnic Themelet

This directory contains a tiny WordPress theme that renders the Model Context Polytechnic site as a static page.

Install `model-context-polytechnic-themelet/` into `wp-content/themes/` and activate it like a normal theme. WordPress gets the expected theme files, `wp_head()`, `wp_footer()`, enqueued styles, and local assets; visitors get the same focused HTML/CSS admissions site without any page-builder or database dependency.

The themelet is intentionally separate from the MCP plugin. Use it when the site itself should look like the Model Context Polytechnic campus while the plugin handles the MCP server and course APIs.

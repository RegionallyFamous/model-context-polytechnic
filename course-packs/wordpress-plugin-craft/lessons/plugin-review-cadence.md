A serious review has passes. First: does the plugin contract match the goal? Second: can unauthorized users write or read sensitive data? Third: is data modeled sanely? Fourth: does it load too much or query too much? Fifth: is it usable and translatable? Sixth: can we test and ship it? Source spine: https://developer.wordpress.org/plugins/developer-tools/helper-plugins/

Style matters, but only after behavior, security, and maintainability are safe.
